The dataset includes log files collected via mcelog, with each file named by the serial number of the DIMM. The data contains 21 columns, such as CPU ID, channel ID, DIMM ID, rank ID, device ID, bank group ID, bank ID, row ID, column ID, and more. These columns provide detailed information about the DRAM errors and system configurations.

failure_ticket.csv includes the failures of each DIMM in this dataset, with columns for serial number, failure time, and server type.

submission.csv requires predictions for DIMM failures, including the serial number, server type, and prediction timestamp. Multiple timestamps are allowed for each DIMM.

Note that we have anonymized the DIMM manufacturer, part number, and other sensitive information to ensure privacy.

For more details, please refer to dataset details.
